# {{ page.title }}

我们课程组的主要成员来自武汉大学计算机学院，主要研究方向是代码分析和机器学习。具体包括：面向新型存储技术的编译优化、运行时优化（内存管理、虚拟机、垃圾回收器）、操作系统级优化（页面调度、文件系统），以及语音识别技术。

# 研究方向
- 面向新型存储技术的软件优化方法
  - 页面调度算法
  - 垃圾回收算法
  - 堆分配算法
- 面向深度学习的编译优化方法
  - 多面体优化
  - 软硬件结合优化
- 语音识别
